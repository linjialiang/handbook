## 加速远程连接

远程客户端链接 mysql 服务器如果遇到需要几十秒才能链接成功，通常是缺少 `skip_name_resolve` 的缘故

```conf
[mysqld]
skip_name_resolve
```

skip_name_resolve 功能说明：

-   不解析主机名，所有主机名都是 IP 或 localhost
-   可有效为远程客户端链接 mysql 服务器提速
